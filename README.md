A modern, responsive Age Calculator built as part of an internship task. This project features a visually appealing UI with animations, glassmorphism design, and dynamic age group classification.

🚀 Features

✨ Dark Glassmorphism UI
A modern blurred-glass interface with elegant dark theme.

🌌 Animated Particle Background
Interactive floating particles for a premium visual experience.

📅 Accurate Age Calculation
Displays age in:

Years
Months
Days

👥 Smart Age Group Classification
The app automatically categorizes users into age groups:

👶 1–18 Years → Pink Theme
Quote: Cute & motivational for youth
🧑 18–40 Years → Yellow Theme
Quote: Motivation for adults
👴 40–100 Years → Blue Theme
Quote: Wisdom & experience-based inspiration

🎬 Smooth Animations
Beautiful transitions and UI effects for better user experience.

📱 Fully Responsive Design
Works perfectly on mobile, tablet, and desktop devices.

🛠️ Tech Stack
HTML5
CSS3 (Glassmorphism + Animations)
JavaScript (Age Logic & Interactivity)
Particle.js / Canvas (for background animation)


🎯 How It Works
User enters their date of birth
JavaScript calculates exact age:
Years
Months
Days
System detects age group
UI theme changes dynamically based on age category
Displays a motivational quote
